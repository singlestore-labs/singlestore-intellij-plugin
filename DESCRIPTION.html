Integrates the SingleStore language server into JetBrains IDEs to provide SQL intelligence for .s2db.sql files.

Features

  • Schema-aware completions — table, column, and procedure suggestions powered by your live SingleStore engine.
  • Syntax highlighting & diagnostics — grammar-based highlighting and error reporting even without a database connection.
  • TCP and WebSocket transport — connects to the language server over a plain TCP socket or a WebSocket (including TLS); the protocol is inferred automatically from the address you enter.
  • Live configuration updates — server address and database credential changes take effect immediately without restarting the IDE.
  • Custom file icon — .s2db.sql files get a SingleStore icon in the Project tree and editor tabs.
